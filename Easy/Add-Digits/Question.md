258. **Add Digits**

Given an integer num, repeatedly add all its digits until the result has only one digit, and return it.<br>

Example 1:<br>
Input: num = 38<br>
Output: 2<br>
Explanation: The process is<br>
38 --> 3 + 8 --> 11<br>
11 --> 1 + 1 --> 2 <br>
Since 2 has only one digit, return it.<br>

Example 2:<br>
Input: num = 0<br>
Output: 0<br>

Constraints:<br>
0 <= num <= 231 - 1
