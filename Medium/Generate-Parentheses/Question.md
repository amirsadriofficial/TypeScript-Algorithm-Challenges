22. **Generate Parentheses**

Given n pairs of parentheses, write a function to generate all combinations of well-formed parentheses.<br>

 

Example 1:<br>
Input: n = 3<br>
Output: ["((()))","(()())","(())()","()(())","()()()"]<br>

Example 2:<br>
Input: n = 1<br>
Output: ["()"]<br>

Constraints:<br>
1 <= n <= 8
